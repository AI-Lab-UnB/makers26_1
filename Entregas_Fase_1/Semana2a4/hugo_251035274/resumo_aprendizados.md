# Aprendizados do Curso: MIT 6.100L - Introduction to CS and Programming Using Python

Este curso é a introdução fundamental do MIT para o pensamento computacional e à programação. Diferente de tutoriais de sintaxe, o foco aqui é em **como pensar como um bom programador** para resolver problemas reais.

---

## 1. Pensamento Computacional e Lógica
O curso ensina que programar não é apenas digitar código, mas criar receitas lógicas (algoritmos) que o computador pode executar.
* **Conhecimento Declarativo vs. Imperativo:** A diferença entre saber o que é um resultado e saber o processo para chegar a ele.
* **Controle de Fluxo:** Uso de condicionais (`if`, `elif`, `else`) e loops (`while`, `for`) para guiar a lógica do programa.


## 2. Decomposição, Abstração e Funções
Como gerenciar a complexidade de grandes sistemas.
* **Abstração:** Esconder detalhes de implementação e focar no que a função faz, não em como ela faz.
* **Decomposição:** Dividir o problema em módulos independentes e reutilizáveis.
* **Escopo de Variáveis:** Entender como as variáveis vivem e morrem dentro e fora das funções (Global vs. Local).

## 3. Algoritmos de Busca e Ordenação
Um dos pilares do curso é o aprendizado de algoritmos clássicos para entender eficiência:
* **Bisection Search (Busca Binária):** Reduz o tempo de busca drasticamente em listas ordenadas.
* **Algoritmos de Aproximação:** Uso do método de Newton-Raphson para encontrar raízes.
* **Recursão:** Uma técnica onde uma função chama a si mesma para resolver subproblemas (ex: Fibonacci, Fatorial).


## 4. Testes e Depuração (Debugging)
Aprender a lidar com o erro é parte fundamental do aprendizado:
* **Exceções:** Uso de `try/except` para lidar com comportamentos inesperados.
* **Asserções (`assert`):** Garantir que as premissas do código sejam verdadeiras durante o desenvolvimento.
* **Estratégias de Debugging:** Testes unitários e teste de caixa preta/caixa de vidro.

## 5. Programação Orientada a Objetos (POO)
O curso introduz a criação de novos tipos de dados.
* **Classes e Instâncias:** Como criar classes para objetos.
* **Métodos e Atributos:** Comportamentos e características dos objetos.
* **Herança:** Criação de hierarquias de classes para reutilizar código e lógica.

## 6. Eficiência Computacional (Notação Big O)
Introdução à análise de algoritmos para entender quão rápido um programa escala conforme os dados aumentam:
* **O(1), O(log n), O(n), O(n^2), O(2^n).**
* Diferença entre eficiência de tempo e eficiência de espaço.
