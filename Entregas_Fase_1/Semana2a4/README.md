# Aprendizados do Curso MIT 6.100L

## Problem Sets

### 1. Lógica Financeira

- **Aprendizado:** Implementação de lógica aritmética para cálculos de juros e poupança.
- **Detalhamento:** Familiarização da Linguagem, uso de **ints** e **floats** para precisão numérica

### 2. Hangman - Manipulação de Strings e Estruturas de Dados

- **Aprendizado:** Construção de um jogo da forca usando a biblioteca random.
- **Detalhamento:** Manipulação de strings, consulta em listas de palavras (alfabeto e letras adivinhadas), tratamento de entradas e lógica de estados das variáveis.

### 3. TF-IDF

- **Aprendizado:** Implementação do algoritmo _Term Frequency-Inverse Document Frequency_, base para determinar importância de palavras em documentos e algoritmos de Machine Learning.
- **Detalhamento:** Manipulação de arquivos de texto, limpeza de dados, uso de dicionários e normalização de dados.

### 4. Estruturas de Dados e Criptografia

- **Aprendizado:** Implementação de Árvores Binárias e verificação de Heaps. Implementação de um algoritmo de criptografia (Cifra de César e One-Time Pad).
- **Detalhamento:**
  - **Árvores:** Construção dos nós, função de altura e verificação de Heaps.
  - **POO:** Introdução básica de Programação Orientada a Objetos, herança de classes (**Message** / **PlaintextMessage** / **EncryptedMessage**) e encapsulamento.

### 5. Manipulação de Imagens

- **Aprendizado:** Manipulação de pixels utilizando a biblioteca Pillow (PIL) e NumPy.
- **Detalhamento:**
  - **Matrizes:** Leitura de imagens através de matriz, transformando imagens em grades de pixels (Colorido ou Preto e Branco).
  - **Filtros:** Uso de multiplicação de matrizes para transformar cores (Simulação de Daltonismo).
  - **Esteganografia:** Manipulação de bits para encontrar mensagens/imagens dentro de imagens.

## Finger Exercises

- **Lógica de Programação:** Treino de sintaxe e lógica em loops, condicionais e estruturas de dados.
- **Estruturas Python:** Familiarização profunda com **list** (listas mutáveis), **tuple** (listas imutáveis) e **dict** (chave-valor).
- **Classes:** Exercícios de familiarização do conceito de classes, utilizando as propriedades de Programação Orientada à Objetos.
- **Complexidade Computacional (Big O):**
  - Análise de eficiência de algoritmos.
  - Identificação de ordens de crescimento: $O(1)$, $O(\log n)$, $O(n)$, $O(n^2)$, $O(2^n)$.
