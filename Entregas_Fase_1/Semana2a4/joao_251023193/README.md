## Fingers
Os fingers são microtarefas interessantes e divertidas para fazer, eles relembram algumas ferramentas e comandos que podemos utilizar nos códigos maiores.

## Problem Set 1
Aprendi a fazer busca binária (bissection search) em Python! Confesso que me deu trabalho, pois já considero complicado fazer a interpretação de um problema na língua materna, agora ter que interpretar bem o texto de uma segunda língua dá o dobro de trabalho. Mas acabou que eu consegui!

## Problem Set 2
Mais complicado que o primeiro, porém mais divertido também. Tive que pesquisar bastante para conseguir fazer o código da forma correta e ainda consegui relembrar algumas ferramentas como o *set()*, que consiste em criar um conjunto e remove as duplicatas (muito importante para o sistema de pontuação).

O processo é interessante, pois eu vou criando o código e quando tenho dúvida em algo ou não sei o conteúdo, eu revejo a aula referente à minha dificuldade. O exemplo desse problem set foi o dicionário: uma ferramenta muito poderosa e com vários métodos. Foi difícil mas prazeroso de ver ele funcionando completo

## Problem Set 4

## Problem Set 5
